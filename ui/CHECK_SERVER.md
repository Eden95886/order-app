# 개발 서버 실행 방법

## 문제 해결 가이드

1. 터미널을 열고 ui 폴더로 이동:
```bash
cd C:\cursor_test\order-app\ui
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 오류 메시지가 나타나면 확인:
   - 모듈을 찾을 수 없다는 오류 → `npm install` 실행
   - 포트가 이미 사용 중 → 다른 포트로 자동 변경됨
   - 기타 오류 → 오류 메시지를 확인

4. 서버가 시작되면 브라우저에서 접속:
   - 기본 주소: http://localhost:5173
   - 터미널에 표시된 주소를 사용하세요
